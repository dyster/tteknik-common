Kopiera ned zip-filen till en mapp på din hårddisk. Packa upp. Om du redan har prislistan installerad kommer den att raderas vid första start setup.exe. Andra start av setup.exe kommer att installera den nya prislistan. Din rabattfil raderas inte vid avinstallationen så om du installerar i samma mapp som den gamla prislistan så kommer din rabattfil att dyka upp igen i nya programmet. Om du installerar i ny mapp så kopiera filen rabattkod tågab.txt till den mappen.
